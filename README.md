## ⚡ Как да използвате

1. Изберете папка с интересуващия ви алгоритъм.  
2. Отворете `.cpp` файла във вашия компилатор или IDE (VSCode, CLion, Visual Studio).  
3. Компилирайте и стартирайте примера.
